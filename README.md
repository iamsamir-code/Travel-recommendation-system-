# AI-Based Travel Recommendation System (CLI)
## Project Description
This is a Command Line Interface (CLI) based travel recommendation system developed using Python. The system suggests travel destinations based on user preferences such as budget, type of place, season, group type, and religious preference.

## Requirements
* Python 3.x
* Command Prompt / Terminal

## Setup Instructions
1. Download or clone the repository
2. Open the project folder in terminal
3. Make sure Python is installed

## How to Run the Project

1. Open terminal in the project folder
2. Run the following command:
python MainBYOP.py

## How to Use the Program
After running the program, the following menu will appear:
1. Recommend
2. View Places
3. View History
4. Exit

### 1. Recommend
Enter:
* Budget (low/medium/high)
* Type (beach/mountains/spiritual/nature/heritage)
* Season (summer/winter/monsoon/all)
* Group (solo/friends/family)
* Religious preference (yes/no)
If "yes", enter your faith.

The system will display recommended places with score.

### 2. View Places
Displays all available places.

### 3. View History
Displays previous recommendations and stored user history.

### 4. Exit
Closes the program.
